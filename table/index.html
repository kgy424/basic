표 만들기

html에서 표를 구현할 때 사용하는 태그

  • tr(table row):행
  • td(table data):열
  • th(table heading):행, 열의 제목(볼드체, 가운데 정렬)
  • 속성 colspan="2":2칸의 열 병합
  • 속성 rowspan="3"3칸의 행 병합
  • colspan, rowspan 적용 후에 원래 있던 열과 행은 주석처리나 삭제 해 줘야함.
